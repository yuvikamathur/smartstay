# SmartStay 🏨

A modern and responsive Hotel & Guest House Booking Website built using React.js, Vite, and Tailwind CSS.

## 📌 Overview

SmartStay is a modern web application designed for hotels and guest houses. It provides an attractive user interface for showcasing rooms, amenities, and gallery images while delivering a smooth user experience across all devices.

## 🚀 Features

* Modern and Responsive Design
* Interactive Navigation Bar
* Hero Section with Call-to-Action
* Room Showcase Section
* Gallery Section
* Mobile-Friendly Layout
* Fast Performance with Vite
* Clean and Professional UI

## 🛠️ Tech Stack

### Frontend

* React.js
* Vite
* JavaScript (ES6+)
* HTML5
* CSS3
* Tailwind CSS
* React Router DOM

## ⚙️ Installation

1. Clone the repository

git clone <repository-url>

2. Navigate to project folder

cd smartstay/client

3. Install dependencies

npm install

4. Start development server

npm run dev
